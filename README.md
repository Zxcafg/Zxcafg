- 🌱 Obecnie uczę się **Zaawansowanej kwantyzacji modeli (QAT), destylacji wiedzy, programowania akceleratorów NPU**

- 👯 Szukam współpracy przy **Optymalizacji modeli głębokiego uczenia maszynowego pod kątem FPGA/układów RISC-V z NPU**

- 🤝 Szukam pomocy w **Zaawansowanych technikach regulacji (MPC) zintegrowanych z sygnałami z sieci neuronowej do śledzenia obiektów**

- 💬 Zapytaj mnie o **systemy wbudowane, STM32, ESP32, C, C++, Python, TensorFlow Lite, Edge AI, Docker, PostgreSQL**
